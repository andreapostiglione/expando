Expando

Text expander open-source per macOS. Trigger → snippet. Ovunque. Solo in locale.

Scarica DMG GitHub Documentazione Hub Marketplace

🔒 Privacy-first

Niente account, niente cloud, niente telemetry. I tuoi snippet restano sul Mac.

⌨️ System-wide

Funziona in browser, Slack, Terminal, IDE — ovunque digiti testo.

📝 YAML + UI

Power user con YAML; editor grafico e hub di package per tutti gli altri.

🔄 Auto-update

Feed Sparkle compatibile; controllo aggiornamenti dal menu bar.

Installazione rapida

Scarica Expando.dmg dalla pagina release, trascina in Applicazioni, poi:

expando setup
expando start

Homebrew

Formula (build da sorgente + CLI):

brew tap andreapostiglione/tap
brew install expando

Cask (DMG precompilato):

brew install --cask andreapostiglione/tap/expando
